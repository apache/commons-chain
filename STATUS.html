1. INTRODUCTION

This scope of the Chain of Responsibility component is to support modelling of complex computations as composition of individual commands that operate on a context.

2. DEPENDENCIES

The Chain of Responsibility component is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • Jakarta Commons Beanutils (OPTIONAL)
  • Jakarta Commons Collections (OPTIONAL)
  • Jakarta Commons Digester (OPTIONAL)
  • Jakarta Commons Logging (OPTIONAL)
  • JavaServer Faces API (Version 1.0 or later) (OPTIONAL)
  • Portlet API (Version 1.0 or later) (OPTIONAL)
  • Servlet API (Version 2.2 or later) (OPTIONAL)
  • JUnit Testing Framework (OPTIONAL, unit tests only)
